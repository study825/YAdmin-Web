## YAdmin-后台管理系统-WEB
> YAdmin管理后台，包含登录、权限管理、站内信等功能。项目代码简练，逻辑清晰明了。可以基于该管理后台，开发"业务系统"，节省开发时间。

#### 依赖
- Node v12.x
- Vue
#### 运行方式(一)
- 运行命令： `npm run dev`
- 打包命令： `npm run build:prod`
